# Basic Forks and C Style Development – Assignment

This project includes examples demonstrating:
- Process creation using `fork()`
- Starting, stopping, pausing, and resuming processes in Linux
- Role of the Linker
- Role of the Loader
- Use of `Makefile` to compile C programs

## Contents
- `process_creation.c` — Demonstrates fork()
- `file1.c` and `file2.c` — Demonstrate linking multiple files
- `simple_program.c` — For loader inspection using `ldd`
- `Makefile` — Builds all programs

## How to Compile
```
make
```

## How to Run
```
./process_creation
./link_example
./simple_program
```
